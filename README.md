# IT23546134_ITPM

IT3040 – ITPM Assignment 1 (Playwright Automation)
=====================================================

This repository contains automated Playwright test scripts developed for IT3040 – ITPM Assignment 1.

System Under Test (SUT)
------------------------

* Website: Swift Translator

Scope
--------

* **Functional Testing**: Verification of Singlish to Sinhala translation functionality with both positive and negative test cases.
* **UI Behavior Testing**: Verification of UI behavior for real-time translation updates and input clearing functionality.

Exclusions
------------

* Backend API, performance, and security testing are excluded as they are out of scope for the assignment.

Test Coverage
-------------

* Positive Functional Test Cases: 24
* Negative Functional Test Cases: 10
* UI Test Cases: 1
* Total Automated Scenarios: 35

Technology Stack
----------------

* **Test Runner**: Playwright
* **Language**: JavaScript (Node.js)

Prerequisites
-------------

* Node.js (LTS version recommended)
* npm (Node package manager)

Check your versions:
node -v
npm -v
